# 洒水器手动触发模组 - 使用指南

## 快速开始

### 1. 安装模组

1. 确保已安装 [SMAPI](https://smapi.io/)
2. 将模组文件夹复制到 `Stardew Valley/Mods/` 目录
3. 启动游戏

### 2. 基本使用

在游戏中：
1. 找到任意洒水器（普通/优质/铱制）
2. **右键点击**洒水器
3. 洒水器会立即浇灌周围的土壤
4. 可以看到喷水动画和听到音效

就这么简单！

## 配置模组

### 方法一：使用 Generic Mod Config Menu（推荐）

1. 安装 [Generic Mod Config Menu](https://www.nexusmods.com/stardewvalley/mods/5098)
2. 在游戏主菜单点击 **选项** → **模组选项**
3. 找到 **Instant Sprinkler** 并点击
4. 调整各项设置

### 方法二：手动编辑配置文件

1. 打开 `Mods/InstantSprinklerMod/config.json`
2. 修改配置项（见下文）
3. 保存文件并重启游戏

## 配置项说明

### 基础设置

#### EnableAnimation（启用动画）
```json
"EnableAnimation": true
```
- `true`：显示喷水动画 ✨
- `false`：不显示动画

#### EnableSound（启用音效）
```json
"EnableSound": true
```
- `true`：播放浇水音效 🔊
- `false`：静音

### 平衡性设置

#### RequireStamina（需要体力）
```json
"RequireStamina": false
```
- `true`：每次触发消耗体力 💪
- `false`：不消耗体力（默认）

**建议**：如果你觉得免费使用太"作弊"，可以开启此选项。

#### StaminaCost（体力消耗）
```json
"StaminaCost": 2.0
```
- 范围：`0.0` - `10.0`
- 默认：`2.0`
- 说明：每次触发消耗的体力值

**参考**：
- 使用锄头：2 体力
- 使用水壶：2 体力
- 使用镐子：2 体力

#### OncePerDay（每天一次）
```json
"OncePerDay": false
```
- `true`：每个洒水器每天只能手动触发一次 📅
- `false`：无限制使用（默认）

**建议**：如果你想保持游戏平衡，建议开启此选项。

### 调试设置

#### DebugMode（调试模式）
```json
"DebugMode": false
```
- `true`：在 SMAPI 控制台显示调试信息
- `false`：不显示调试信息（默认）

**用途**：仅在遇到问题时开启，用于排查错误。

## 推荐配置方案

### 方案一：休闲模式（默认）
```json
{
  "EnableAnimation": true,
  "EnableSound": true,
  "RequireStamina": false,
  "StaminaCost": 2.0,
  "OncePerDay": false,
  "DebugMode": false
}
```
**特点**：无限制使用，适合休闲玩家。

### 方案二：平衡模式
```json
{
  "EnableAnimation": true,
  "EnableSound": true,
  "RequireStamina": true,
  "StaminaCost": 2.0,
  "OncePerDay": true,
  "DebugMode": false
}
```
**特点**：消耗体力且每天限制一次，保持游戏平衡。

### 方案三：极简模式
```json
{
  "EnableAnimation": false,
  "EnableSound": false,
  "RequireStamina": false,
  "StaminaCost": 2.0,
  "OncePerDay": false,
  "DebugMode": false
}
```
**特点**：无动画无音效，快速浇水。

## 常见问题

### Q: 为什么点击洒水器没有反应？

A: 请检查：
1. 是否使用**右键**点击（不是左键）
2. 是否有打开的菜单或对话框
3. 洒水器周围是否有可浇水的土壤
4. 如果开启了"每天一次"，是否已经使用过

### Q: 如何知道洒水器已经使用过了？

A: 如果开启了"每天一次"限制，再次点击会显示红色提示：
- 中文：`这个洒水器今天已经使用过了。`
- 英文：`This sprinkler has already been used today.`

### Q: 体力不足会怎样？

A: 如果开启了"需要体力"且体力不足，会显示红色提示：
- 中文：`体力不足，无法使用洒水器。`
- 英文：`Not enough stamina to use the sprinkler.`

### Q: 支持哪些洒水器？

A: 支持所有原版洒水器：
- 普通洒水器（十字形，4格）
- 优质洒水器（3x3，8格）
- 铱制洒水器（5x5，24格）
- 以上洒水器 + 加压喷头（范围扩大）

### Q: 支持花盆吗？

A: 是的！支持室内花盆（IndoorPot）。

### Q: 多人游戏可以用吗？

A: 可以！每个玩家都需要安装此模组。

### Q: 会影响游戏平衡吗？

A: 默认配置下，此模组可以视为"作弊"。如果你想保持平衡：
1. 开启"需要体力"
2. 开启"每天一次"限制
3. 适当调整体力消耗值

## 技巧和建议

### 💡 技巧 1：早晨快速补浇
如果忘记种植作物就睡觉了，第二天早上可以手动触发洒水器补浇。

### 💡 技巧 2：温室管理
在温室中，可以随时手动触发洒水器，方便种植新作物。

### 💡 技巧 3：配合加压喷头
加压喷头会扩大洒水范围，手动触发时也会生效！

### 💡 技巧 4：节省时间
在大型农场中，手动触发洒水器比用水壶逐个浇水快得多。

## 卸载模组

1. 删除 `Mods/InstantSprinklerMod` 文件夹
2. 重启游戏

**注意**：卸载模组不会影响存档，洒水器会恢复正常的过夜浇水功能。
