# Instant Sprinkler Mod

一个星露谷物语模组，允许玩家通过右键点击洒水器来手动触发洒水，无需等待第二天。

## 功能特性

- ✨ **即时洒水**：右键点击洒水器即可立即浇水，无需等待过夜
- 🎨 **视觉反馈**：播放原版喷水动画和音效
- 🔧 **高度可配置**：支持多种配置选项
- 🌍 **多语言支持**：支持中文和英文
- 🎮 **兼容性强**：
  - 支持所有类型的洒水器（普通、优质、铱制）
  - 支持加压喷头附件
  - 支持花盆（IndoorPot）
  - 兼容 Stardew Valley 1.6+

## 安装方法

1. 安装 [SMAPI](https://smapi.io/)
2. 下载本模组
3. 将模组文件夹解压到 `Stardew Valley/Mods` 目录
4. 运行游戏

## 使用方法

1. 在游戏中，右键点击任意洒水器
2. 洒水器会立即浇灌其覆盖范围内的所有土壤
3. 可以看到喷水动画和听到浇水音效

## 配置选项

安装 [Generic Mod Config Menu](https://www.nexusmods.com/stardewvalley/mods/5098) 后，可以在游戏内配置以下选项：

- **启用动画**：是否显示喷水动画（默认：是）
- **启用音效**：是否播放浇水音效（默认：是）
- **需要体力**：是否消耗体力（默认：否）
- **体力消耗**：每次触发消耗的体力值（默认：2.0）
- **每天一次**：是否限制每个洒水器每天只能手动触发一次（默认：否）
- **调试模式**：是否在控制台显示调试信息（默认：否）

也可以手动编辑 `config.json` 文件来配置。

## 技术细节

### 支持的洒水器类型

| 洒水器类型 | 基础范围 | 加压喷头后 |
|----------|---------|-----------|
| 普通洒水器 | 十字形（4格） | 3x3（8格） |
| 优质洒水器 | 3x3（8格） | 5x5（24格） |
| 铱制洒水器 | 5x5（24格） | 7x7（48格） |

### 兼容性

- ✅ 支持 Stardew Valley 1.6+
- ✅ 支持多人游戏
- ✅ 支持花盆（IndoorPot）
- ✅ 支持加压喷头（Pressure Nozzle）
- ✅ 集成 Generic Mod Config Menu

### 实现原理

本模组基于 SMAPI 的事件驱动架构：

1. 监听 `Input.ButtonPressed` 事件
2. 检测玩家是否右键点击了洒水器
3. 计算洒水器的覆盖范围（考虑加压喷头）
4. 修改覆盖范围内土壤的湿度状态（`HoeDirt.state = watered`）
5. 播放喷水动画和音效

详细的技术文档请参考 `洒水器手动触发模组文档.txt`。

## 平衡性说明

默认配置下，本模组不消耗体力且不限制使用次数，这在某种程度上可以视为"作弊"。如果您希望保持游戏平衡，建议：

- 启用"需要体力"选项
- 启用"每天一次"限制
- 适当调整体力消耗值
